java使用jdbc连接mysql的代码

在此使用的是Class.forName(driver)的办法加载jdbc，当然也可以使用Driver driver = new com.mysql.jdbc.Driver()创建driver,不过使用前者可以将数据库解耦出来，因此更好一些，我们可以通过配置文件的形式随时切换数据库。

对于使用Class.forName不用加载driver类，是由于JDBC规范中明确要求这个Driver类必须向DriverManager注册自己，因此这一步已经在加载时做到了。
